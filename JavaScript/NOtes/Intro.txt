Comments in JS:
    Comments are run by any compiler or engines. THey are used for readibility of the code.In JS, we can write 2
    types of comments. 
        Single Line: These comments are followed after double slashes "//" These are used for only single line.
        Multi Line: These comments are within /* */ these are used for single or multiple lines.
